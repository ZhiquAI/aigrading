# 评分细则示例

本文件夹包含了各种题目的 JSON 格式评分细则示例，可以直接导入到 AI 阅卷助手中使用。

## 📁 文件列表

- `15-资产阶级革命综合题.json` - 第15题，涉及英国资产阶级革命、美国独立战争、法国大革命的综合题，共14分

## 📖 使用方法

### 在扩展中导入

1. 打开 AI 阅卷助手扩展
2. 点击"配置评分细则"
3. 选择"导入 JSON"
4. 选择对应的 JSON 文件即可

### JSON 格式说明

每个评分细则都遵循以下结构：

```json
{
  "version": "2.0",
  "questionId": "题号",
  "title": "题目类型",
  "totalScore": 总分,
  "scoringStrategy": {
    "type": "评分类型",
    "strictMode": false,
    "allowAlternative": true,
    "openEnded": false
  },
  "answerPoints": [
    {
      "id": "得分点ID",
      "content": "答案内容",
      "keywords": ["关键词1", "关键词2"],
      "requiredKeywords": ["必须关键词"],
      "score": 分值,
      "openEnded": false,
      "deductionRules": "扣分规则"
    }
  ],
  "gradingNotes": [
    "阅卷提示1",
    "阅卷提示2"
  ]
}
```

## 🎯 评分策略类型

- **pick_n**: 任选 N 个得分点（如"任答3点得满分"）
- **all**: 必须答全所有得分点
- **weighted**: 加权评分（不同得分点有不同分值）

## ⚙️ 关键功能

- ✅ **关键词匹配** - 支持灵活的关键词识别
- ✅ **必须关键词** - 确保核心概念必须出现
- ✅ **扣分规则** - 明确的评分标准
- ✅ **时空围栏** - 防止跨时代错误
- ✅ **逻辑验证** - 确保答案逻辑一致性
- ✅ **开放性题目** - 支持灵活评分

## 📝 创建自己的评分细则

您可以：
1. 复制现有示例作为模板
2. 修改评分点、关键词等内容
3. 使用扩展的"生成评分细则"功能自动创建

## 💡 提示

- JSON 文件使用 UTF-8 编码
- 建议使用有意义的文件名（如题号-题目类型.json）
- 可以在 gradingNotes 中添加时空围栏和逻辑验证规则
